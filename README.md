# Cloud Collaborative System - Backend

## Overview
This is the backend of the Cloud Collaborative System, a real-time collaborative coding platform. It provides authentication, project management, real-time communication, and code execution services.

## Features
- User Authentication
- Project Management
- Real-time Collaboration
- Docker-based Code Execution
- REST APIs
- MongoDB Database

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Socket.IO
- Docker
- JWT Authentication

## Installation

1. Clone the repository
2. Run:
   npm install
3. Start the server:
   node server.js
